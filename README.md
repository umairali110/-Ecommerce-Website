# 🛒 Full Stack E-Commerce Platform

A production-ready full-stack e-commerce application built with modern technologies.

## 🚀 Tech Stack
- Frontend: Next.js, React, Tailwind CSS, Redux, Axios
- Backend: NestJS, TypeORM, PostgreSQL
- Authentication: JWT Role-Based System
- Payments: Stripe Integration
- Image Storage: Cloudinary

## ✨ Features
- User authentication (Login/Register + OTP)
- Role-based access (Admin / User)
- Product management system
- Cart system (Add / Update / Remove)
- Order placement system
- Inventory management
- Admin dashboard
- Cloudinary image upload
- Stripe payment integration

## 📦 Project Structure
- ecommerce-frontend
- ecommerce-backend
